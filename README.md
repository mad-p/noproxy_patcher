# Noproxy Patcher

Apply monkey patch to rubygems and bundler so that they respect NO_PROXY environment variable.

## Usage

    noproxy_patcher [-n] [-q]

noproxy_patcher applies NO_PROXY patches rubygems and bundler that resides under your ~/.rvm.
